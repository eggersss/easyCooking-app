## üíª Run Locally

Clone the project

```bash
  git clone git@github.com:tiemifaustino/easycooking-app.git
```

Go to the project directory

```bash
  cd easycooking-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
